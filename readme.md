# Team Task Manager

## Overview
Team Task Manager is a full-stack web application that allows teams to manage projects, assign tasks, and track progress with role-based access control.

## Features
- User Authentication (Signup/Login)
- Role-Based Access Control (Admin/Member)
- Project Management
- Task Creation & Assignment
- Task Status Tracking
- Dashboard with Task Statistics

## Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- Database: SQLite
- Deployment: Railway

## Installation

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

## Deployment
The application is deployed on Railway.
